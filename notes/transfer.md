## Remaining validation

To fully validate the extraction, the pipeline needs to be run end-to-end:
1. Run render_free_recall.ipynb → regenerates HealeyKahana2014 figures
2. Run render_serial_recall.ipynb → regenerates Gordon2021 figures
3. Run render_parameter_shifting.ipynb → regenerates shifting figures
4. Run model comparison notebooks → regenerates tables/
5. Run rename_figures.py → produces submission/ from generated tifs
6. Diff submission/ against monorepo's indexed_figures/
7. `cd manuscript && quarto render` → clean manuscript build
